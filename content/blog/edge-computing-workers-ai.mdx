## AI at the Edge

Cloudflare's [Workers AI](https://developers.cloudflare.com/workers-ai/) lets you run inference models directly on the edge. No GPUs to manage, no cold starts for model loading.

### Use Cases

We're exploring three primary use cases:

1. **Content moderation**: Scan user submissions in real-time
2. **Language detection**: Route requests to the right locale
3. **Smart caching**: Predict which content to pre-warm

### Benchmarks

Running Meta's Llama 3.2 1B model on Workers AI:

| Metric | Value |
|--------|-------|
| Cold start | `<1ms` |
| Inference time | 45ms |
| First token | 120ms |

### Limitations

There are real constraints. Model sizes are limited to 2GB, and not all architectures are supported. For heavy workloads, we still fall back to dedicated inference endpoints. But for lightweight edge tasks, Workers AI is a game-changer.

The edge is getting smarter every quarter.
