## Why Web Vitals Matter

Google uses Core Web Vitals as a ranking signal. But beyond SEO, they directly impact user experience and conversion rates.

### Our Vitals Scorecard

| Metric | Target | Ours |
|--------|--------|------|
| LCP | < 2.5s | 0.4s |
| FID | < 100ms | 12ms |
| CLS | < 0.1 | 0.02 |

### How We Got There

**LCP**: Pre-rendered HTML from SSG means the browser gets content immediately. No JavaScript needed for the first paint.

**FID**: Minimal JavaScript on initial load. We defer interactive components until after the page is fully rendered.

**CLS**: Explicit dimensions on all images, and no layout shifts from dynamically injected content.

### The LLM Factor

Beyond traditional crawlers, LLM scrapers (GPTBot, Claude, Gemini) also parse HTML directly. A static HTML-first approach means your content is always visible — to both humans and machines.
